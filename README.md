# Python-SQL-Ecommerce-Project
**E-commerce Sales Analysis: Automated data pipeline and business intelligence insights on Brazilian market trends.**

This is an end-to-end data analytics pipeline designed to derive actionable business insights from a large-scale Brazilian e-commerce dataset containing over 100,000 orders (2016-2018). The workflow covers data extraction, cleaning, storage in a MySQL database, and advanced analytical reporting.

**Key Features:**

**Data Engineering:** Automated the extraction and loading of multi-relational CSV files into MySQL using Python.

**Analytical SQL:** Performed deep-dive analysis on customer behavior, regional distribution, and seller revenue rankings.

**Advanced Metrics:** Applied window functions and complex joins to calculate year-over-year growth, customer retention rates, and moving averages.

**Visualization:** Developed insightful charts and trends using Matplotlib and Seaborn to present findings effectively.

**Technologies Used:**

Python (Pandas, Matplotlib, Seaborn)

SQL (MySQL)

Jupyter Notebook
